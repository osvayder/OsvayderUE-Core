## Capture Rules

- Do not show private filesystem paths, user names, auth tokens, sessions, or unreleased project names.
- Prefer cropped editor windows over full desktop captures.
- Keep logs short and use only reviewer-relevant markers.
- Do not claim a video exists until the file is captured and added.
